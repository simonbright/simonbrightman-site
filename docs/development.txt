Local development
=================

Install dependencies:
  npm install

Start full local server (static site + TTS audio function):
  npm run dev

Wait for "Local dev server ready" then open the site on port 8888 in your browser.

Static-only preview (no article audio):
  npm run dev:static
  Opens on port 3000.

Build CSS after HTML changes:
  npm run build

Deploy preview (not production):
  npm run deploy:draft

Deploy production (simonbrightman.com):
  npm run deploy

Domain notes
------------
Primary: simonbrightman.com
WWW redirects to apex. HTTP redirects to HTTPS.
DNS: A record for apex to Netlify load balancer IP, CNAME www to netlify subdomain.

Adding articles
---------------
1. Edit source markdown in insights_package/NN_slug/canonical_name.md
2. Create or update insights/your-slug/index.html (use /images/*.png for diagrams)
3. Update data/insights.json and insights/index.html inline JSON
4. npm run build
